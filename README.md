Práctico Sprint 5:
Objetivos:
Dotar una aplicación web que demuestre el empleo de los contenidos desarrollados en el módulo backend, comprendidos en node.js, Express, Mongoose. La cual:
*Consume los datos de una api externa: https://restcountries.com/v3.1/region/america.
*Procesa, filtra y guarda en MongoDB sólo los datos de los países hispanohablantes.
*Muestra un dashboard web con la implementación de los métodos CRUD.
*Dispone un layout que contiene una views la misma posee layout, dashboard, index, partials en el que se desarrolla el navbar y el footer. Tiene validaciones backend.
Tecnologías usadas:
           *Node.js. 
            
            *MongoDB.
            
            *Mongoose.
            
            *EJS.
            
            *CSS.
            
            *JavaScript.
            
            *SweetAlert2. 
            
            *express.
            
            *express-ejs-layouts.
            
            *express-validator.
            
            *method-override.
            
            *dotenv.
            
            *node-fetch.


Pasos de ejecución:
Instalar el proyecto:
*clonar el repositorio: gh repo clone Mar-Jose/Nodo-FS.3-Sprint-5.
Instalar dependencias:
                 *mongoose.
                  
                  *express.
                  
                  *express-ejs-layouts.
                  
                  *express-validator.
                  
                  *method-override.
                  
                  *dotenv.
                  
                  *node-fetch.


Ejecutar el proyecto:

                 *Abrir el navegador: http://localhost:3000/api/paisesA.
